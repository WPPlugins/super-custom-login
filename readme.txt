This plugin will customize your login screen. It replaces the default WordPress logo with your own super custom logo.

== Description ==

Customize your login screen. Replace the WordPress logo with your custom logo and make it link to your homepage. Enhance security by removing error messages upon failed login attempts. A template file for your logo is included.

Use the included template to ensure your custom logo fits properly. Your custom logo should be saved as a tranparent PNG named "login_logo.png", and must be located in your theme's images directory. 

Example: YourDomainName.com/wp-content/themes/YourTheme/images/login_logo.png

== Installation ==

1. Uncompress super-custom-login.zip
2. Open login_logo.psd in Photoshop (or graphic editing program of choice) and customize logo.
3. Save as transparent PNG named login_logo.png
4. Upload entire folder (except image files) to the /wp-content/plugins/ directory.
5. Upload login_logo.png to the /wp-content/themes/YourTheme/images/ directory.
6. Activate the plugin through the 'Plugins' menu in WordPress.
